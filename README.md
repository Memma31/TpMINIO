# TpMINIO
TP Cloud 1 : Introduction au Cloud avec OpenTofu et MinIO

Ce Travaux Pratiques (TP) vise à introduire les concepts fondamentaux de l'**Infrastructure as Code (IaC)** et du **Stockage Objet** en utilisant des outils open source :

* **OpenTofu :** Alternative libre à Terraform, utilisée pour gérer de manière déclarative l'infrastructure.
* **MinIO :** Solution de stockage objet auto-hébergée, compatible avec l'API AWS S3.

L'objectif final est de déployer un **site web statique** sur MinIO en gérant la configuration (variables/secrets) et en optimisant les règles de sécurité (ACLs et Politiques S3).
